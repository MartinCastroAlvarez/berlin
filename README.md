## Instructions

#### Installing dependencies
```bash
sudo npm install -g expo-cli
sudo npm install -g create-react-native-app
yarn install
```

#### Running unit tests
```bash
npm run test
```

#### Running on Android
```bash
npm run android
```

#### Running on iOS
```bash
npm run ios
```

#### Running on web
```bash
npm run web
```

#### Running on web
```bash
npm run start
```
